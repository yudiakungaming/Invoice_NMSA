    ⚙️Konfigurasi Firebase

    Firebase Configuration

    Butuh Firestore saja (Gratis, tanpa kartu kredit).

    ❌Jangan copy ini:
    // Import statements...
    const firebaseConfig = {
      apiKey: "...",
      authDomain: "...",
    };
    const app = initializeApp(config);
    const db = getFirestore(app);
    ✅Copy HANYA object config:
    {
      "apiKey": "AIzaSy...",
      "authDomain": "your-app.firebaseapp.com",
      "projectId": "your-project-id",
      "storageBucket": "...",
      "messagingSenderId": "...",
      "appId": "..."
    }
    2

    Cara Kerja File → Google Drive

    1
    User upload file → Dikonversi jadi Base64 string
    2
    Data + Base64 disimpan ke Firebase Firestore
    3
    Google Apps Script trigger → Buat file di Google Drive
    4
    Status sync update real-time di aplikasi

    Batasan:

    • Maksimal 750KB per file
    • File muncul di Drive dalam 5-10 menit
    • Tidak perlu kartu kredit!
