Setup Integrasi

⚠️ Pastikan URL di-deploy sebagai "Web app" dengan akses "Anyone"

Cara Deploy yang Benar

  1. Buka Google Apps Script Editor
  2. Paste kode script (sudah saya sediakan)
  3. Klik Deploy → New deployment
  4. Pilih type: "Web app" (bukan API!)
  5. Execute as: Me
  6. Who has access: Anyone ⚠️
  7. Klik Deploy
  8. Copy URL yang muncul, paste di atas

🛠️ Panduan Perbaikan Koneksi

Masalah yang Terjadi

Gagal terhubung ke Google Apps Script. Ini biasanya karena konfigurasi deployment yang salah.

📋 Langkah Perbaikan (Ikuti Urutan):

1

Buka Google Apps Script Editor

Klik link ini untuk membuka editor:

https://script.google.com/macros/s/AKfycbwNLf2l8nncu8CfztY1EFyymSelDXrIletn1QN-SJiqdaFwA52nkPBEfcOqyOBLFjHW/edit

Hapus Semua Kode Lama

Tekan Ctrl+A (semua), lalu Delete

3

Paste Kode Baru

Copy kode di bawah, paste ke editor:

// Finance Sync Script v2.0
// PASTE KE GOOGLE APPS SCRIPT EDITOR

const SPREADSHEET_ID = '13jou3fy-pwpVxKbPRCAI0bB1cj3JlK5olXrYEnWcH9s';
const SHEET_NAME = 'Sheet1';

function doGet(e) {
  return ContentService.createTextOutput(JSON.stringify({
    status: "ok", message: "Webhook aktif",
    timestamp: new Date().toISOString()
  })).setMimeType(ContentService.MimeType.JSON);
}

function doPost(e) {
  try {
    const data = JSON.parse(e.postData.contents);
    const ss = SpreadsheetApp.openById(SPREADSHEET_ID);
    let sheet = ss.getSheetByName(SHEET_NAME);
    if (!sheet) sheet = ss.insertSheet(SHEET_NAME);
    
    if (sheet.getLastRow() === 0) {
      sheet.appendRow(['No','Tanggal','Lokasi','Kode','Jenis','Invoice','Items','Qty','Status','Total','TglBayar','Dibayarkan','Catatan','DibuatOleh','Disetujui','Keuangan','DirKeu','DirUtama','Accounting','Timestamp','Source']);
      sheet.getRange(1,1,1,21).setFontWeight(true).setBackground('#1a2642').setFontColor('#fff');
      sheet.setFrozenRows(1);
    }
    
    let items = "";
    if(data.items) items = data.items.map(i => i.ket+" ("+i.qty+") - Rp "+i.nominal).join("; ");
    
    sheet.appendRow([
      sheet.getLastRow(), data.tanggal||'', data.lokasi||'',
      data.kode||'', data.jenis_pengajuan||'', data.no_invoice||'',
      items, '-', data.status||'', data.total_nominal||0,
      data.tanggal_pembayaran||'', data.dibayarkan_kepada||'',
      data.catatan_tambahan||'', 'Nur Wahyudi', 'Harijon',
      'Andi Dhiya Salsabila', 'Harijon', 'H. Andi Nursyam Halid',
      'Sri Ekowati', new Date().toISOString(), 'FinanceSync'
    ]);
    
    return ContentService.createTextOutput(JSON.stringify({
      success:true, row:sheet.getLastRow()
    })).setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService.createTextOutput(JSON.stringify({
      success:false, message:err.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}
4

Deploy sebagai Web App (PENTING!)

  1. Klik menu Deploy → New deployment
  2. Klik icon gear ⚙️ (Select type)
  3. Pilih "Web app"
  4. Execute as: Pilih "Me"
  5. Who has access: Pilih "Anyone" ⚠️
  6. Klik Deploy
⚠️ Jika pilih "Authorized users only" → AKAN GAGAL!
5

Authorize Permission (Izin Akses)

Setelah klik Deploy, akan muncul popup:

  1. Klik "Authorize access"
  2. Pilih akun Google Anda
  3. Klik "Advanced" (lanjutan)
  4. Klik "Go to [nama project] (unsafe)"
  5. Klik "Allow"
6

Copy URL Web App Baru

Setelah sukses deploy, copy URL yang muncul:

https://script.google.com/macros/s/[NEW-ID]/exec
7

Test Koneksi di Aplikasi

  1. Tutup modal ini
  2. Klik tombol ⚙️ Settings di header
  3. Paste URL baru di field Webhook
  4. Klik "Simpan Konfigurasi"
  5. Klik "Test Koneksi"
  6. Jika hijau ✅ → BERHASIL!

Butuh Bantuan Visual?

Cari di YouTube: "Google Apps Script Deploy Web App Anyone"

Ada banyak tutorial video yang menunjukkan langkah-langkah persis seperti di atas.

😕 Masih Gagal? Coba Ini:

  • Buka file HTML pakai Chrome (bukan Edge/Firefox)
  • Clear cache browser (Ctrl+Shift+Delete)
  • Disable extensions (ad blocker, dll)
  • Coba incognito window (Ctrl+Shift+N)
  • Redeploy script (hapus deployment lama, buat baru)
  • Share spreadsheet ke email yang sama dengan script
