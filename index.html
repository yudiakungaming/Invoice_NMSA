    🔥 Firebase Configuration

    Hanya butuh Firestore (Gratis, tanpa kartu kredit).

    ❌ Jangan copy ini:
    // Import...
    const firebaseConfig = {
      apiKey: "...",
      ...
    };
    const app = initializeApp(...);
    ✅ Copy HANYA ini:
    {
      "apiKey": "AIzaSy...",
      "authDomain": "...",
      "projectId": "...",
      ...
    }
    2

    📖 Cara Kerja File → Google Drive

    1️⃣
    User upload file → Dikonversi jadi Base64
    2️⃣
    Data + Base64 disimpan ke Firebase Firestore
    3️⃣
    Apps Script membuat file di Google Drive & dapat link shareable
    4️⃣
    Status otomatis update di aplikasi (real-time listener)

    ⚠️ BATASAN:

    • Maksimal 750KB per file (limit Firestore document)
    • File muncul di Drive setelah 5-10 menit (Apps Script trigger)
    • Tidak perlu kartu kredit!
    • Status sync otomatis ter-update via real-time listener
